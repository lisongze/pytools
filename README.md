# Tools
## code2pdf
### *Description*
A tool by python which convert c++ code source file to pdf file.
### *Install*
```sudo apt install enscript```

```pip install glob```
### *Usage*
```python code2pdf.py ${SOURCE_CODE_ROOT} ${OUT_PDF_ROOT}```
